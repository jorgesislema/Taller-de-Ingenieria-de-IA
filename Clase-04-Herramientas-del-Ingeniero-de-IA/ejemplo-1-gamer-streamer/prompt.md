# Prompt para reproducir el Ejemplo 1 — Gamer / Streamer

> Copia este prompt entero y pegalo en ChatGPT, Claude o Gemini.
> Reemplaza `[Tu Nombre]` con tu nombre o apodo antes de enviarlo.

---

```markdown
## Rol
Eres un diseniador de portfolios para GitHub especializado en perfiles de gamers, streamers y creadores de contenido jovenes (12 a 20 anios). Tu estilo es fresco, moderno, con emojis bien usados y lenguaje que un adolescente usaria.

## Contexto
Soy [Tu Nombre o Apodo], un chico/chica de [tu edad] anios que esta aprendiendo IA y programacion. Me gustan los videojuegos ([menciona tus juegos favoritos: Valorant, Minecraft, Roblox, LoL, Fortnite, etc.]), estoy empezando a crear contenido en [Twitch / YouTube / TikTok] y quiero aprender a hacer bots para Discord y herramientas para streamers.

Estoy cursando un Taller de Especializacion en IA y esta es mi primera tarea: crear mi portfolio en GitHub.

## Tarea
Genera un archivo README.md para mi perfil de GitHub. El README va a ser mi carta de presentacion.

## Restricciones
- Formato: Markdown listo para copiar y pegar en GitHub
- Tono: juvenil, fresco, con personalidad, pero no infantil. Como habla un streamer de 18 anios.
- Secciones obligatorias:
  1. Titulo con emoji y subtitulo descriptivo
  2. "Sobre mi" (2-3 parrafos)
  3. "Que estoy aprendiendo" (lista de skills con emojis)
  4. "Proyectos" (al menos 3, con descripcion y herramientas usadas)
  5. "El setup" (tabla con herramientas, uso y nivel de 1 a 4 estrellas)
  6. "Encontrame" (links a redes: GitHub, Discord, Twitch/YouTube)
  7. Frase final o "quote" inspiradora
- Usa emojis: 👾 🎮 🕹 ⚡ 🎯 🤖 🎬 🧠 📦 📡
- No uses datos falsos especificos. Si no doy un dato, usa placeholders tipo [tu-canal] o [tu-usuario].
- Los proyectos que menciones deben sonar realistas para un adolescente (bot de Discord, clips, calculadora de ELO, paginita web simple, scripts para stats de juegos).
- Maximo 400 palabras.

## Ejemplo del formato que quiero
# 👾 [Nombre] — Level Up
**Gamer | Streamer en formacion | Explorando IA para crear bots y mods**
---
## 🕹 Sobre mi
[parrafos...]
## 🎯 Que estoy aprendiendo
[lista con emojis...]
## ⚡ Proyectos
[3 proyectos con descripcion...]
## 🎒 El setup
[tabla...]
## 📡 Encontrame
[links...]
> "frase final..."
```

---

### Como usar este prompt

1. Reemplaza `[Tu Nombre o Apodo]` y `[tu edad]` con tus datos reales
2. Agrega tus juegos favoritos donde dice `[menciona tus juegos favoritos...]`
3. Copia todo el bloque de codigo de arriba
4. Abri ChatGPT, Claude o Gemini
5. Pega el prompt
6. La IA te va a generar un README.md personalizado
7. Copia el resultado y pegalo en tu archivo `README.md`
8. Si algo no te gusta, pedile ajustes: "cambia el tono", "agrega mas emojis", "quitale X"

> **Tip:** Si no tenes 3 proyectos reales, no importa. La IA te va a sugerir ideas realistas basadas en tu perfil. Despues vos las ajustas a tu realidad.
