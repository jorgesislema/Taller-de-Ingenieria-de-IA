# Clase 4 — Herramientas del Ingeniero de IA + Vibe Coding: Tu Primer Repositorio con IA

**Modulo 1: Infraestructura y Limitaciones de la IA**

---

## Resumen

Hoy ponemos todo junto. Ya sabes que es la IA (Clase 1), entendes sus riesgos (Clase 2) y sabes cuanto cuesta usarla (Clase 3). Ahora viene la parte practica: **vas a crear tu primer repositorio de GitHub usando inteligencia artificial como tu asistente**.

En el camino vas a aprender a crear agentes personalizados en ChatGPT, Claude y Gemini. Vas a instalar las herramientas que todo ingeniero de IA usa: VSCode, Git (terminal y GitHub Desktop), Markdown y Opencode. Y repasamos los fundamentos de prompt engineering (agentes, skills, loops).

Al final de esta clase vas a tener **tu propio repo de portada** publicado en GitHub, generado enteramente con ayuda de IA. Tu primer proyecto real.

---

## Objetivos

Al terminar esta clase vas a poder:

1. Repasar los fundamentos de prompt engineering: elementos, tipos, agentes, skills y loops.
2. Explicar que es el Vibe Coding y por que cambia las reglas del juego.
3. Crear agentes personalizados en ChatGPT, Claude y Gemini.
4. Instalar y usar VSCode, Git (CLI + GitHub Desktop), Markdown y Opencode.
5. Usar IA para generar tu primer repositorio de GitHub: una pagina de portada profesional.
6. Publicar tu repo y compartir la URL.
